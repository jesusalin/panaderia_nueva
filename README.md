# Sistema de Gestión de Inventario - Panadería Muruwasi

## Descripción

Este proyecto consiste en el desarrollo de un sistema web para la gestión de inventarios de la Panadería Muruwasi.

La aplicación permite administrar ingredientes, proveedores, compras, ventas y usuarios mediante una plataforma web desarrollada utilizando Laravel y MySQL.

El objetivo principal es optimizar el control de inventarios y mejorar la gestión de la información dentro de la organización.

## Funcionalidades

### Gestión de Usuarios

Permite registrar y administrar usuarios del sistema.

### Gestión de Ingredientes

Permite registrar, editar, eliminar y consultar ingredientes.

### Gestión de Proveedores

Permite administrar proveedores de la panadería.

### Gestión de Compras

Control de compras realizadas a proveedores.

### Gestión de Ventas

Registro y control de ventas realizadas.

### Dashboard Administrativo

Visualización de información relevante para la gestión del negocio.

## Instalación

1. Clonar el repositorio

git clone URL_DEL_REPOSITORIO

2. Instalar dependencias

composer install

3. Configurar archivo .env

4. Crear la base de datos en MySQL

5. Ejecutar migraciones

php artisan migrate

6. Ejecutar servidor

php artisan serve
